# AAN_leaf_disease_detection
Deep learning-based potato leaf disease detection using CNN on the PlantVillage dataset, inspired by IEEE 2025 research for accurate and efficient classification of Early Blight, Late Blight, and Healthy leaves
# Potato Leaf Disease Detection using CNN

##  Overview
Potato is one of the most widely cultivated crops, but its production is highly affected by diseases such as Early Blight and Late Blight. Traditional disease detection methods rely on manual inspection by experts, which is time-consuming, error-prone, and not scalable.

This project proposes an automated potato leaf disease detection system using Convolutional Neural Networks (CNN). The model is trained on image data to accurately classify potato leaves into different disease categories. The work is inspired by a 2025 IEEE research paper focusing on efficient and accurate plant disease detection using deep learning.


##  Objectives
- To develop an automated system for potato leaf disease detection  
- To classify leaf images into Early Blight, Late Blight, and Healthy categories  
- To reduce human effort and improve detection accuracy using deep learning  

---

##  Features
- Automatic detection of potato leaf diseases  
-  CNN-based deep learning model  
-  Trained on PlantVillage dataset  
-  High accuracy and efficient performance  
-  Simple and user-friendly implementation  


## Dataset
- **PlantVillage Dataset (Kaggle)**
- Contains labeled images of potato leaves:
  - Early Blight  
  - Late Blight  
  - Healthy  
- Images are preprocessed and resized (e.g., 256×256×3)  
- Data augmentation is applied to handle class imbalance  



##  Methodology
1. Data Collection from PlantVillage dataset  
2. Image Preprocessing (resizing, normalization)  
3. Data Augmentation (rotation, flipping, etc.)  
4. CNN Model Design (5-layer architecture)  
5. Model Training and Validation  
6. Performance Evaluation  
7. Disease Prediction and Visualization  



## Results
- The model achieves high classification accuracy on test data  
- Effectively distinguishes between healthy and diseased leaves  
- Provides fast and reliable predictions
